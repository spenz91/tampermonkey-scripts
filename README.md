# Tampermonkey Scripts

A collection of Tampermonkey userscripts. Click an install link below to add a script — it will auto-update whenever a new version is pushed.

> Requires the [Tampermonkey](https://www.tampermonkey.net/) browser extension.

## Scripts

| Script | Description | Install |
|---|---|---|
| [AK3 Auto Scan](ak3-autoscan/) | Automates the AK3 scanner setup workflow on IWMAC plant servers | [Install](https://raw.githubusercontent.com/spenz91/tampermonkey-scripts/main/ak3-autoscan/AK3-Autoscan.user.js) |
| [Rocketlane Enhancer](rocketlane-enhancer/) | Hides the Gantt calendar and adds a floating chat panel on Rocketlane | [Install](https://raw.githubusercontent.com/spenz91/tampermonkey-scripts/main/rocketlane-enhancer/rocketlane-enhancer.user.js) |
| [Rocketlane Project Notes](rocketlane-project-notes/) | Adds a writable Note column to the Rocketlane Projects list with SQL persistence and live clickable URL links | [Install](https://raw.githubusercontent.com/spenz91/tampermonkey-scripts/main/rocketlane-project-notes/rocketlane-project-notes.user.js) |
| [Oneflow + HubSpot Copy Products](oneflow-copy-products/) | Adds copy buttons on Oneflow (tilbud PDF products) and HubSpot (deal Line items card) — both copy as a bulleted list with bold headers | [Install](https://raw.githubusercontent.com/spenz91/tampermonkey-scripts/main/oneflow-copy-products/oneflow-copy-products.user.js) |
